# 🛒 CodeAlpha EDA — Retail Store Inventory Analysis

> An interactive Exploratory Data Analysis (EDA) web app built with **Python**, **Streamlit**, and **Plotly** to analyze and visualize retail store inventory data.

---

## 📌 Overview

This project performs a complete **Exploratory Data Analysis** on a retail store inventory dataset. It includes data cleaning, statistical summaries, and rich interactive visualizations — all accessible through a user-friendly Streamlit web interface.

## 📁 Project Structure

```
CodeAlpha_EDA/
│
├── app.py                          # Main Streamlit application
├── EDA.ipynb                       # Jupyter Notebook (exploration & analysis)
├── retail_store_inventory.csv      # Clean dataset
├── retail_store_inventory_messy.csv# Raw / messy dataset
└── requirements.txt                # Python dependencies
```

---

## 🔍 Features

- 📊 **Interactive charts** with Plotly (bar, pie, histogram, scatter, etc.)
- 🧹 **Data cleaning** pipeline on the messy dataset
- 📈 **Statistical summaries** (mean, median, std, missing values, etc.)
- 🔎 **Filter & explore** data dynamically through the UI
- 📱 Fully responsive and deployable on Streamlit Cloud

## ⚙️ Installation & Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/youssefalini/CodeAlpha_EDA.git
cd CodeAlpha_EDA

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```

---

## 📊 Dataset

The dataset contains retail store inventory records including:
- Product names & categories
- Stock levels & quantities
- Prices & sales data
- Store locations

Two versions are provided:
- `retail_store_inventory.csv` — cleaned version
- `retail_store_inventory_messy.csv` — raw version with noise & missing values
